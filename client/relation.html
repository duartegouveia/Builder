Choosing a correlation method:

  • Pearson (r) - Linear relationship. Assumes normality. Best for continuous data. Range: -1 to +1.
  • Spearman (ρ) - Monotonic relationship. No normality assumption. Robust to outliers. Range: -1 to +1.
  • Kendall's Tau (τ) - Rank correlation. Better for small samples (n<30) or ordinal data. More robust than Spearman. Range: -1 to +1.
  • Point-Biserial (rpb) - When one variable is binary (0/1) and other is numeric. Range: -1 to +1.
  • Phi (φ) - Association between two binary variables. Special case of Pearson for 2×2 tables. Range: -1 to +1.
  • Cramér's V - Categorical association. Based on chi-squared. Range: 0 to 1 (no direction).

Auto-detect logic:

  • Both binary → Phi
  • One binary + one numeric → Point-Biserial
  • Both numeric, n<30 → Kendall's Tau
  • Both numeric, n≥30 → Pearson
  • Either categorical → Cramér's V
